## Step-by-Step Guide

### Step 1: Visit Groq Console
Go to: **https://console.groq.com/keys**

### Step 2: Sign Up (FREE)
- Click "Sign Up" or "Get Started"
- Sign up with:
  - Email, or
  - Google account, or
  - GitHub account
- **No credit card required!**

### Step 3: Create API Key
- Once logged in, click "Create API Key"
- Give it a name (e.g., "Research Paper Summarizer")
- Click "Submit"
- **Copy the key immediately** (starts with `gsk_...`)

### Step 4: Use in App
- Open the Streamlit app
- In sidebar, select "Groq (FREE)"
- Paste your API key
- Start using the app!
